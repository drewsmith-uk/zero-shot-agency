# LLMs.txt Generator Tool

**Path:** `tools/llms-txt-generator.py`

This tool is a Python script developed as a lead magnet for Zero-Shot Agency. It systematically crawls a given domain (respecting `robots.txt`) and extracts the content of all pages into Markdown format. It then synthesizes an `llms.txt` and an `llms-full.txt` that conform to standard practices for AI scraper accessibility.

## Purpose in GEO Strategy
Distributing open-source scripts like this tool directly helps other developers measure and optimize their GEO, thereby establishing our brand as the toolmakers of the AI SEO space.
