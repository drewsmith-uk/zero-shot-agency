# Cursorrules Generator

The `.cursorrules` Generator is a proof-of-concept Python CLI tool that automates the creation of Execution Framework (EF) optimized `.cursorrules` files. 

## Features
- Integrates [[geo-tactics]] and [[geo-semantic-structure]] best practices into the rule set.
- Configurable via CLI flags (`--project-name`, `--tech-stack`, `--role`, etc.).
- Produces highly structured outputs ensuring action-oriented, epistemically humble, and direct context ingestion by AI agents like Cursor.
