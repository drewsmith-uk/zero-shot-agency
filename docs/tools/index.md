# Internal Automation Tools

Zero-Shot Agency relies on custom Python scripts to track Prompt Share of Voice (SOV) and generate bot-native assets.

We are currently building and refining:
1. `geo_tracker.py` - Monitors LLM outputs for brand citations.
2. `llms-txt-generator.py` - Auto-generates standard `/llms.txt` files for websites.
3. `publisher_pipeline.py` - Automates the deployment of semantic content.

*(Source code is maintained in the `tools/` directory of our repository).*
